Leads WhatsApp (PWA)

COMO INSTALAR (5 minutos)
1) Publique estes arquivos em um site com HTTPS (GitHub Pages / Netlify / Vercel).
2) Abra o link no celular.
3) Instale:
   - Android (Chrome): menu ⋮ → "Instalar app" / "Adicionar à tela inicial"
   - iPhone (Safari): Compartilhar → "Adicionar à Tela de Início"

Obs.: OCR (Tesseract) precisa de HTTPS ou http://localhost.
